# Android Debug Bridge
O Android Debug Bridge (ADB) é uma ferramenta de linha de comando que permite a comunicação entre um dispositivo Android e um computador. O ADB é parte do Android SDK e é utilizado para depurar aplicativos, instalar APKs, transferir arquivos, e muito mais.

> `adb` está incluído no pacote Android SDK Platform Tools.
