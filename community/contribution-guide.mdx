## Por que Contribuir?
Open-source work is never done, and neither is documentation. Contributing to docs is a good way for beginners to get involved in open-source and for experienced developers to clarify more complex topics while sharing their knowledge with the community.

By contributing to the Mobile Docs, you're helping us build a more robust learning resource for all developers. Whether you've found a typo, a confusing section, or you've realized that a particular topic is missing, your contributions are welcomed and appreciated.

## Como Contribuir
O conteudo da Documentação pode ser encontrado no repositório da [@hackingforce/mobile-docs](https://github.com/hackingforce/mobile-docs). Para contribuir, você pode editar os arquivos diretamente no GitHub ou clonar o repositório e editar os arquivos localmente.

## Escrevendo em MDX
A Documentação é inteiramente escrita em [MDX](https://mdxjs.com/), uma sintaxe que combina Markdown com JSX. Leia o [Github Markdown Guide](https://mdxjs.com/) para aprender mais sobre como escrever usando markdown.
